# PeggleEdit
A WYSIWYG level editor for Peggle Nights written in C# / WinForms.

![media/screenshots/environment.png](media/screenshots/environment.png)

## Quick Start
- Download the peggleedit.zip [here](https://github.com/IntelOrca/PeggleEdit/releases/latest).
- Make a folder, then extract all the files from that zip into the folder.
- Open peggleedit.exe.
- Use the Pack Explorer to add/open levels, then use the Tools tab to place pegs, bricks, paths, objects, and generators.
- Save the pack as a `.pak` file and put it in the Peggle Nights `levelpacks` folder to play it.

## Art Overlay
Peggle levels use 800x600 backgrounds. PeggleEdit can crop, move, and resize imported backgrounds in the Set Background screen, including a toggle for the in-game UI overlay. You can also use this [template image](docs/img/bg_template.png) in another image editor if you prefer.
